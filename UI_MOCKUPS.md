## User Flow Summary

### Adding a New Domain:

1. **Click "Add Domain"** in page header
2. **Step 1**: Enter domain or select suggestion â†’ Click "Next"
3. **Step 2**: 
   - Either: Check "I have added the CNAME record"
   - Or: Click "Connect DNS Automatically" (if supported) and complete Domain Connect flow
4. **Click "Submit"** â†’ Modal closes, domain appears in table with "In Process" status
5. **Auto-refresh** updates status every 5 seconds:
   - Success â†’ "Registered/Completed"
   - Failure â†’ "Error" (with retry button)

### Viewing Domain Details:

1. Click "Details" button on any domain row
2. Modal opens showing:
   - Current status and timestamps
   - Apple Merchant ID
   - Cloudflare information
   - CNAME configuration
   - Error messages (if any)
3. Click "Close" to return to table

### Retrying Failed Domain:

1. Locate domain with "Error" status
2. Click "Retry" button
3. System attempts onboarding again
4. Status updates via auto-refresh
